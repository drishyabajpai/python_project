## ▶️ Getting Started

Clone this repository:

```bash
git clone https://github.com/drishyabajpai/python_project.git
```

Go to the project folder:

```bash
cd python_project
```

Run any Python file:

```bash
python filename.py
```

Example:

```bash
python contact_book.py
```
